Como funciona a criação de um site?

A criação de um site, a primeira vista, parece algo extremamente complexo a primeira vista, inúmeros códigos e dialetos na qual, a primeira vista, é incompreensível, porém, ao observarmos e estudarmos a superficialidade, já conseguimos compreender todo o funcionamento do site.


Para um site ser criado, utilizamos diversas linguagens de programação categorizadas em 2. Front-End e Back-End.


Back-End

O Back-End é a parte da programação que fica por trás de todo o site, são os “bastidores” da programação. Php, Python, Java, JavaScript, .NET (C# e VB), e outras como PHP, Ruby e SQL são algumas das linguagens que são Back-End, elas estão responsáveis por todo o funcionamento do site, como o armazenamento dos dados obtidos em um banco de dados (SQL) e a ligação entre a parte visual com o mesmo armazenados (PHP), a interatividade do site (JavaScript) entre outros.



Front-End

O Front-End é a parte da frente da programação, a “apresentação” do site, toda a parte visual é feita pelas linguagens responsáveis por essa área, sendo elas as que nós vamos te mostrar nesse site!



Vamos começar com o CSS.


O CSS é uma linguagem de estilização, ou seja, ela é responsável por embelezar o seu site usando a sua base como principal referência, o HTML.


Quando você cria um site, primeiramente o HTML é responsável por fazer o seu esqueleto. Toda a base do site, o conteúdo, as imagens, os títulos, todos são feitos pelo HTML.


Logo depois que essa base está finalizada, nós introduzimos o CSS no nosso site.


O CSS ele abrirá portas para você fazer diversas estilização no seu site. Isso deixará ele muito mais elegante e agradável aos olhos do seu usuário, o que facilitará o entendimento de tudo o que você quis passar para o seu cliente, melhorando completamente a experiência do usuário caso fosse apenas o HTML bruto.


Essa linguagem ela funciona com diversos comandos colocandos no elemento do HTML ou numa classe ou num ID que definimos a alguma dessas classes, podendo ser estilizado com cores, tamanhos, fontes e animações.

Agora vamos falar um pouco sobre HTML


O HTML, como dito anteriormente ele é responsável pela base de todo o site, tudo o que você enxerga veio dele.


Essa linguagem funciona com diversas tags, tags essas que vão aos poucos se acumulando e dando total forma a sua página.

Ele diz o que vai aparecer na página. Textos, imagens, títulos, botões, vídeos, tabelas, formulários, entre muitas outras coisas

E por fim, vamos falar do JAVA SCRIPT


O Java Script é a linguagem que dá vida ao site.


Como assim dar vida ao site?


Bom, enquanto o HTML é a estrutura e o CSS a aparência, o Java Script ele torna a página muito mais dinâmica, podendo:


  • Mostrar uma mensagem quando alguém clica num botão,
  • Fazer uma imagem mudar sozinha,
  • Validar um formulário (como verificar se o e-mail foi preenchido)
  • Criar animações
  • Mostrar ou esconder coisas sem recarregar a página
  • Fazer jogos, calculadoras, menus interativos

Ela é realmente uma linguagem de programação, diferente do HTML e do CSS


Mas o que isso quer dizer?


Dizer que essa linguagem é uma linguagem de programação, é falar que podemos usar comandos lógicos para desenvolver o código


Como por exemplo os comandos "if" e "else", os comandos "for", entre diversos outros comandos.


Um pequeno detalhe que devemos atentar é que JAVA e JavaScript são diferentes linguagens, mesmo que tenham o nome parecido. Muitas pessoas acabam por confundir devido a esse detalhe.
