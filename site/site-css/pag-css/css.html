Demonstrações

Bem, como visto anteriormente, o CSS é toda a estilização do seu site, da sua página, e aqui, você verá pequenos exemplos do funcionamento dela, com códigos a mostra para você utilizar!


Seletores e IDs


No CSS, usamos seletores para dizer ao navegador quais elementos queremos estilizar.


Um dos seletores mais comuns é o ID, que é identificado com o símbolo # (cerquilha).


Cada ID deve ser único na página e serve para aplicar estilos específicos a um único elemento.


Por exemplo, se um elemento tem o ID "menu", podemos usar #menu no CSS para mudar sua cor, tamanho ou posição.


Além do ID, também podemos usar classes, que são identificadas com um ponto (.) antes do nome. As classes podem ser aplicadas a vários elementos ao mesmo tempo, o que é útil quando queremos que vários itens compartilhem o mesmo estilo.


Combinando esses seletores, conseguimos criar páginas bem organizadas, com estilos personalizados para cada parte da estrutura do site.


Alguns exemplos: 


/* ID: usa-se o símbolo # seguido do nome do ID */
#minhaDiv {
    background-color: blue;
    color: white;
    padding: 10px;
}

/* Classe: usa-se o ponto . seguido do nome da classe */
.caixa {
    border: 1px solid black;
    padding: 20px;
}

/* Elemento: seleciona diretamente pelo nome da tag HTML */
p {
    font-size: 16px;
    line-height: 1.5;
}

/* Combinando seletores: */
div.caixa {
    background-color: lightgray;
}

#minhaDiv p {
    font-weight: bold;
}


Propriedades e estilos comuns


O CSS, como você viu no exemplo acima, ele usa propriedades para dar características a seus elementos, veja alguns deles: 



color → Cor do texto

background-color → Cor de fundo

font-size → Tamanho da letra

font-family → Fonte usada (ex: Arial, Verdana)

text-align → Alinhamento do texto (esquerda, centro, direita)

margin → Espaço externo do elemento

padding → Espaço interno do elemento

border → Borda do elemento

width / height → Largura e altura

Exemplo na prática:


.font-color
{
    color: red;
}

Modelo de caixa

O modelo de caixa é basicamente o que o nome diz: A "caixa" que o seu elemento tem.


Nele temos:


content (conteúdo),

padding (espaço interno),

border (borda)

margin (espaço externo)

Exemplo: 

    
    .card {
      border: 1px solid gray;
      padding: 20px;
      margin-bottom: 10px;
    }

Animações e Transições


A animações e transições no CSS serve para deixar a sua página um pouco mais dinâmica, vamos ver os comandos.


transition → Animação suave entre estilos (ex: cor mudando)

@keyframes → Animações mais complexas

animation-name, animation-duration, etc.

Exemplos


    @keyframes colorir {
        0%{
            color: black;
        }
        25%
        {
            color: red;
        }
        50%
        {
            color: yellow;
        }
        75%
        {
            color: orange;
        }
        100%
        {
            color: white;
        }
    }

    /*Outra parte*/

    #colorirFonte
    {
        animation: colorir 3s infinite alternate;
    }
