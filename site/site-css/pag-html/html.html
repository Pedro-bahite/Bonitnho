TAGS GERAIS

<!DOCTYPE>: Define o tipo de documento e a versão do HTML

<html> Raiz de um documento HTML

<head> Contém metadados/informações para o documento

<meta> Define metadados sobre o documento HTML

<title> Define o título para o documento

<body> Contém o conteúdo visível da página

<style> Usada para incluir informações de estilo CSS.

<script> Usada para incluir ou referenciar um código JavaScript em um documento HTML. 

                        <!DOCTYPE html>
        <html lang="pt-BR">
            <head>
                <meta charset="UTF-8">
                <title>Exemplo de Código</title>
            </head>
            <body>
                <h1>Olá, Mundo!</h1>
                <p>Este é um exemplo simples de código HTML.</p>
            </body>
        </html>

LINKS E IFRAMES

<a> Define um hiperlink

<iframe> Define um quadro embutido (inline frame)

FORMULÁRIOS

<form> Define um formulário para entrada do usuário

<input> Define um campo de entrada

<textarea> Define um campo de texto multilinha

<label> Define uma etiqueta para um elemento <input>

<button> Define um botão clicável

<select> Define uma lista suspensa

<option> Define uma opção dentro de um elemento <select>

<fieldset> Agrupa elementos relacionados em um formulário

<legend> Define um título para o grupo <fieldset>

TABELAS

<table> Define uma tabela

<tr> Define uma linha em uma tabela

<td> Define uma célula em uma tabela

<th> Define uma célula de cabeçalho em uma tabela

<thead> Agrupa cabeçalhos de colunas em uma tabela

<tbody> Agrupa o corpo do conteúdo em uma tabela

<tfoot> Agrupa rodapés de colunas em uma tabela

<caption> Define o título de uma tabela

LISTA

<ul> Define uma lista não ordenada

<ol> Define uma lista ordenada

<li> Define um item de uma lista

<dl> Define uma lista de descrição

<dt> Define um termo na lista de descrição

<dd> Descreve o termo na lista de descrição

DIVISÓRIAS

<div> Define uma divisão ou seção em um documento

<header> Define um cabeçalho para o documento ou seção

<footer> Define um rodapé para o documento ou seção

<nav> Define um conjunto de links de navegação

<section> Define uma seção em um documento

<article> Define um conteúdo independente ou autônomo

<aside> Define conteúdo à parte do conteúdo ao qual está relacionado

<main> Especifica o conteúdo principal do documento
